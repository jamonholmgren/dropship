Dropship

The fast and simple way to deploy your web app to a DigitalOcean Droplet.

An open-source project by Jamon Holmgren.

Why Dropship?

Dropship automates the setup and deployment of your web app to DigitalOcean Droplets, giving you full control without any hidden costs.

  • Built specifically for DigitalOcean Droplets
  • No confusing Docker setup required
  • Easy setup with dropship setup
  • One command deploys: git push live
  • Supports macOS and Linux

CLI Commands:

git push live
dropship setup

Getting Started

To start using Dropship, simply run the following command in your terminal: 

alias dropship="source <(curl -sSL https://dropship.jamon.dev/cli)"

Then, initialize Dropship in your project folder:

dropship setup

Follow the prompts to configure your deployment.
